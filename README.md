# Beraca's Valley (Flutter)

Application Flutter cross-platform pour la gestion d'une salle de reception.

## Lancement

Placez-vous dans `flutter_app` et exécutez :

```bash
flutter pub get
flutter run
```

## Fonctionnalités

- Consultation des chambres
- Consultation du menu
- Réservation en ligne
- Mode administration pour gérer les réservations
